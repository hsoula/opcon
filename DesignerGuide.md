# Units #
Units is modeled as an [entity](entity.md) that contains a [state](state.md) data (unique to each unit), and is a collation of models that are derived from templates. A unit that is the command element of an echelon will be defined by its size and identity as well as by the command\_echelon and the echelon\_name.
